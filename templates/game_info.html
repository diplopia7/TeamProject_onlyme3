스탯설명

통솔 : 플레이어의 통솔 수치가 적장의 통솔보다 낮을 경우 10장을 제외합니다
무력 : 플레이어의 전술 스킬에 영향을 줍니다. 또한 적장의 무력보다 낮을 경우 플레이어는 전술 스킬을 사용할 수 없습니다.
지력 : 플레이어의 전술 스킬에 영향을 줍니다. 또한 적장의 지력보다 낮을 경우 플레이어는 전술 스킬을 사용할 수 없습니다.
정치 : 게임 스토리 진행에 영향을 줍니다.

내정특기설명

상재 : 금 수입 1.5배
경작 : 병량 수입 1.5배
연병 : 모병량 1.5배
수집 : 아이템 제작 및 수집시 능력치+10
인맥 : 부장수 최대+10
변설 : 대실패 제거

전투특기설명

호걸 : 절대 전투에서 죽지 않습니다.
귀모 : 적의 전술 스킬을 무효화합니다.
군사 : 반대 상성 병종도 두배의 피해를 입힐 수 있습니다.
냉정 : 반대 상성 병종에게 피해를 1배로 받습니다.

전술계 스킬 설명

공격강화 : 플레이어의 턴마다 공격카드의 공격력이 +1됩니다.(5번까지 중첩 가능)
방어강화 : 플레이어의 턴마다 방어카드의 방어력이 +1됩니다.(5번까지 중첩 가능)
색적 : 플레이어가 무조건 선공이 됩니다.
속공 : 모든 전투카드들의 공격력과 방어력이 -50%가 됩니다. 두 번 행동할 수 있습니다.
복병 : 마법카드를 서치할 수 있습니다. 덱에 마법카드가 없다면 할 수 없습니다.
복병신속 : 복병의 효과와 공격강화의 효과를 얻습니다. 플레이어의 턴마다 10d를 돌려 6이 나오면 스킵합니다.
노당익장 : 공격카드의 공격력이 +2, 방어카드의 방어력은 -1이 됩니다.
맹공 : 공격카드의 공격력이 +3, 방어카드의 방어력은 -5가 됩니다.
공방일체 : 플레이어의 턴마다 공격카드의 공격력이 +1, 방어카드의 방어력이+1이 됩니다.(5번까지 중첩 가능)
강용 : 방어카드의 방어력이 +2가 됩니다.
철벽 : 방어카드의 방어력이 +3, 공격카드의 공격력이 -2가 됩니다.
강습 : 전병과에 2배 피해를 줄 수 있습니다. 플레이어의 턴마다 10d를 돌려 6이 나오면 스킵합니다.
만인적 : 플레이어의 무력이 +10, 통솔이 +10이 됩니다. 체력 최대치는 50이 됩니다.(부장 효과는 똑같이 적용)
운체풍신 : 플레이어의 턴마다 공격카드의 공격력 +1, 방어카드의 방어력 +1이 됩니다. (3번까지 중첩가능)
통솔+5, 지력+5가 됩니다.
위무지강 : j,k,q를 서치 후, 세팅을 하고 시작합니다. 플레이어 덱에 하나라도 없으면 스킬시전이 불가합니다.
방호 : 방어카드의 방어력이 +10이 됩니다. 공격카드의 공격력은 -5가 됩니다.
의용병 : 자신의 덱을 50% 제외 시킴으로서, 플레이어 자신의 묘지카드 순서를 마음대로 바꿀 수 있습니다.
팔진도 : 필드에 카드 드로우를 두 장 씩 전개가 가능합니다. 두 장씩 드로우 할 때 마다 체력을 20 소모합니다.
용의지혜 : 체력을 99 소모합니다. 적군의 모든 필드를 클리어 합니다. 스페이드 a는 영향을 받지 않습니다.
대타격 : 100d1 주사위를 굴려 플레이어의 무력계수 이하 만큼의 주사위 수만큼 피해를 입힙니다.
(본인 무력 계수 초과 주사위가 나오면 실패)
화계 : 100d1 주사위를 굴려 플레이어의 지력계수 이하 만큼의 주사위 수 만큼 피해를 입힙니다.
(본인 지력 계수 초과 주사위가 나오면 실패)
신화계 : 적에게 매 턴마다 15의 피해를 입힙니다.
인중여포 : 100d1 주사위를 굴려 본인의 무력계수 이하 만큼의 주사위 수 만큼 피해를 입힙니다.
모든 필드 카드를 다 제외시킵니다. 2턴 당 한 번 행동이 가능합니다.

병종 상성

창병>기병>궁병>창병

내정관련

1. 금100 = 병량5000과 교환 가능
2. 부장 한 명당 추가 병력은 1000명 = 추가 체력10
3. 수집(수집 특기가 없을 경우 수집 행동을 할 수 없습니다.)
3-1. 수집 아이템
아이템은 무기, 갑옷, 말, 책서, 장신구 총 5가지 획득 가능합니다.
능력치는 통솔, 무력, 지력, 정치 중 하나를 고를 수 있습니다.
4. 병력 1000 = 플레이어 체력 10 최대체력 이상의 병력을 모집할 수 없습니다.
5. 제작 : 아이템 제작 시도시 금을 200 소모합니다. (10d1 주사위를 굴려 스텟을 정합니다.)
6. 아이템 교체시 금을 200소모합니다. 능력치가 동일한 아이템만 교체가 가능합니다.

게임오버

1. 국가병력이 0이 된 후, 다음 전투 패배 시 플레이어 게임 오버
2. 금, 병량이 모두 0이 되면 게임 오버
3. 플레이어에게 병력, 병량이 없으면 게임오버
